Static export of the Smart Expedite site.

How to view:
- Firefox / Safari: double-click index.html
- Chrome / Edge: serve via local server (ES module security):
    python3 -m http.server 8080
  then open http://localhost:8080

Contents:
- index.html              entry point (uses relative paths)
- assets/                 minified CSS, JS, images, videos
- truck-rate-map.html     embedded interactive map
- robots.txt, favicon.ico
