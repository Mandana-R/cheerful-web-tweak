Cheerful Logistics — Standalone Static Export
=============================================

This folder is a real, no-build website. Open index.html directly in any
browser (double-click works — no server required).

Contents
--------
- index.html, shippers.html, carriers.html, services.html,
  rate-calculator.html, about.html, blog.html, contact.html, account.html
- assets/css/styles.css   — all styling, plain CSS (no Tailwind, no build)
- assets/js/main.js       — header/footer injection, slider, rate
                            calculator, contact + carrier forms, tabs
- assets/img/             — images and videos used by the pages
- favicon.ico, robots.txt, placeholder.svg

No React, no Vite, no routing library, no API calls. Forms and the rate
calculator run entirely in the browser as a demo.

If you also need the original source code, it lives next to this folder
in src/ (TypeScript + React) and is included in the project ZIP.
